# Investor Service Request Management Portal 

## Project Overview 

This project is a service request management platform for a Mutual Fund organization. 

It helps investors raise service requests digitally, track request progress, and receive status updates. It also provides an admin portal for operations teams to manage requests and resolve them within the required time. 

### Features 

--> Mobile Application (Investor App) 

The mobile app is used by investors to manage their service requests. 

# Features: 

Investor Login 
Dashboard 
Raise Service Request 
Bank Detail Update Request 
Nominee Update Request 
Address Update Request 
Track Request Status 
Notifications 

# Request status: 

OPEN 
IN_PROGRESS 
RESOLVED 


--> Web Portal (Admin Portal) 

The web portal is used by service executives and administrators. 

# Features: 

Admin Login 
Dashboard 
View All Requests 
Process Investor Requests 
Update Ticket Status 
SLA Tracking 
Resolution Dashboard 
Ticket workflow: OPEN → IN_PROGRESS → RESOLVED 


--> SLA Tracking 

SLA (Service Level Agreement) is used to monitor request resolution time. 

Example: 

Bank Update → 2 Days 
Nominee Update → 3 Days 
Address Update → 1 Day 
If request is delayed, it is marked as: 
SLA BREACHED
